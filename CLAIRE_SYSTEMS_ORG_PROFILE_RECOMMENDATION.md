# Claire-Systems Organization Profile Recommendation

Use this as the plain-English organization profile after repository cleanup.

```markdown
# Claire Systems

Claire Systems builds governed AI runtime components for memory, provenance, session continuity, and evidence-aware workflows.

## What CLAIRE Is

CLAIRE is a governed AI runtime. It routes user input through memory recall, policy checks, provider/model calls, output validation, and trace logging before returning an answer.

CLAIRE is not just a chatbot. The goal is an AI system that can remember relevant prior context, ignore irrelevant memory, explain what supported an answer, and preserve a trace of what happened.

## What ARE Is

ARE means Analog Recall Engine. ARE is the external memory authority for CLAIRE-style systems.

ARE records experience chronologically, stores durable records outside the model, and preserves provenance through hashes or checksums. Search and vector indexes may help find records, but ARE remains the source of memory authority.

## What Session Continuity Is

Session Continuity is the layer that helps CLAIRE carry useful state across sessions without rewriting history. It reads recalled ARE records, identifies current versus historical facts, preserves corrections, and surfaces unresolved conflicts.

## What Veritas Is

Veritas is the evidence and document-intelligence surface. Veritas workflows organize source documents, preserve source hashes, create traceable evidence records, and prepare review material for humans.

Veritas Legal is for evidence organization and attorney-review support. It is not legal advice and does not replace attorneys.

## Repository Status

Active repositories should be labeled clearly as one of:

- Core
- Demo
- Research
- Prototype
- Legacy
- Private product

Public repositories should include a README, maturity status, license or source-available notice, and clear usage instructions where appropriate.

## Contact

For licensing, acquisition, research collaboration, or technical review, contact the Claire Systems project owner.
```
