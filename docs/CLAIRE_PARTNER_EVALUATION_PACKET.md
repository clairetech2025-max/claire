# Claire Partner Evaluation Packet

## What Claire Is

Claire is a governed cognition layer built around persistent memory, deterministic recall, provenance, policy control, traceability, and orientation before generation.

Claire is designed to sit beside or above enterprise systems, not replace them.

Claire also has prototype evidence for structured operational continuity through Project Lantern Relay, the Session Capsule Protocol room invented by Lucius Prime with Codex build assistance. Session Capsules preserve the end-state of a work session so later work can restart from a verified handoff rather than raw chat history.

## Why It Matters

Most AI systems generate from transient context and retrieved snippets. Claire is designed to orient first: intent, authority, risk, memory lane, provenance, policy, and output mode.

That makes Claire relevant for domains where wrong memory, wrong authority, or missing trace can create real consequences.

Session Capsules add a practical restart layer. Instead of burying operational state inside a long transcript, a capsule preserves what changed, what failed, where the restore point is, what the next safe step is, and what should not be repeated.

## Benchmark Evidence

Drive-supplied benchmark evidence reports sub-millisecond local deterministic recall, HMAC verification, recall + verify, and tamper detection under Android / ARM64 Termux conditions.

This is credible evidence, not a universal performance guarantee.
