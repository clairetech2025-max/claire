# CLAIRE Control Interface

Docker Space package for the CLAIRE governed runtime mirror.

This deployment uses sanitized source from the GitHub branch selected by the
deployment process. It does not include Azure secrets, private runtime memory,
databases, model weights, uploads, indexes, logs, or private legal evidence.

Runtime secrets and provider endpoints must be configured with Hugging Face
Space Secrets and Variables.
