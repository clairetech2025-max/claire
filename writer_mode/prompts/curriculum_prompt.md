# Curriculum Prompt

Create Claire Code Academy curriculum material from approved source notes.

Requirements:

- Teach one idea at a time.
- Use simple examples.
- Do not invent biographical or technical claims.
- Mark gaps as `[NEEDS REVIEW]`.
- Include `Draft for human review`.

Preferred lesson shape:

1. Lesson title.
2. What the student will learn.
3. Plain-English explanation.
4. Practical example.
5. Exercise.
6. Review questions.
7. Next lesson bridge.
