# Story Chapter Prompt

Write a clean markdown chapter draft from the selected transcript lane.

Requirements:

- Preserve Steve's voice.
- Do not invent facts.
- Build from the transcript material only.
- Mark gaps as `[NEEDS REVIEW]`.
- Keep the writing direct, vivid, and controlled.
- Do not make it corporate.
- Do not over-explain Claire architecture unless the source material requires it.
- Include `Draft for human review`.

Preferred structure:

1. A strong opening grounded in a real moment or pressure.
2. The work, conflict, mistake, discovery, or learning curve.
3. What Steve understood afterward.
4. How this lane connects to Claire, if the source supports that connection.
5. Review notes for facts that need confirmation.
